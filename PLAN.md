# PLAN: 과제 24시간 폭탄 진행 모션

## 목표
과제 카드가 D-1로 보일 때 폭탄 레일을 표시하고, 실제 마감 24시간 이하부터 폭탄이 24시간 지점에서 0시간 지점으로 실시간 이동하는 모션을 표시한다.

## 가정
- 폭탄 모션은 과제 목록 카드 안에만 표시한다.
- D-1이지만 실제로는 24시간 초과로 남은 과제에는 폭탄을 시작점에 대기 상태로 표시한다.
- D-2 이상 과제와 이미 마감된 과제에는 표시하지 않는다.
- 실시간 업데이트는 기존 메모 입력을 끊지 않도록 전체 목록 재렌더 대신 폭탄 UI만 갱신한다.

## 접근
1. 남은 시간을 기반으로 폭탄 표시 여부, 진행률, 라벨을 계산하는 헬퍼를 추가한다.
   - verify: D-1 또는 24시간 이하/마감 전 조건에서만 정보가 반환된다.
2. `renderTasks()`가 각 과제 카드에 폭탄 슬롯을 렌더링하고, 조건에 맞을 때만 보이게 한다.
   - verify: D-2 이상 과제에는 폭탄 UI가 화면에 보이지 않는다.
3. 1초 주기 업데이트 함수로 폭탄 위치와 남은 시간 라벨을 갱신한다.
   - verify: 메모 입력 중에도 카드 전체가 재생성되지 않는다.
4. CSS로 폭탄 레일, 라벨, 움직임을 구성하고 모바일에서도 넘치지 않게 한다.
   - verify: 폭탄 UI가 카드 안에 안정적으로 배치된다.
5. 캐시 버전 갱신, 문법 검증, 코드 리뷰 후 커밋/푸시한다.
   - verify: `node --check script.js`, `node --check sw.js`, `codex review --uncommitted` 통과.
