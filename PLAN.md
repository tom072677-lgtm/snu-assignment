# PLAN: 식당 길찾기 누락 보강

## 목표
식당 탭의 모든 표시 가능한 식당 상세에서 길찾기 버튼이 나오도록 좌표 매칭을 보강한다. 기존 인앱 카카오 지도 경로 표시 흐름은 유지한다.

## 가정
- 사용자가 말한 "어떤 식당"은 식당 상세 패널에서 길찾기 버튼이 표시되지 않는 항목이다.
- 서버가 제공하는 고정 식당과 SNUCO 학식 세부 식당 모두 대상이다.
- 정확 좌표가 없는 식당은 앱 내 기존 지도 데이터 중 가장 가까운 대표 위치로 매칭한다.

## 접근
1. 현재 식당 목록과 지도 좌표 연결 방식을 확인한다.
   - verify: `getRestaurantLoc`와 `SNU_LOCATIONS`의 누락 케이스를 식별한다.
2. 고정 식당 ID 매칭을 추가한다.
   - verify: `boodang` 같은 고정 식당에도 길찾기 버튼이 생성된다.
3. SNUCO 세부 식당명 매칭을 별칭/정규화 기반으로 보강한다.
   - verify: 학생회관/220동, 자하연, 공대, 두레, 예술계, 감골 등 이름 변형에도 좌표가 나온다.
4. 문법 검증과 가능하면 코드 리뷰를 수행한다.
   - verify: JavaScript 구문 오류가 없고 `codex review --uncommitted` 결과를 확인한다.
