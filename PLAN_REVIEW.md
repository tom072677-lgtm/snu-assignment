# PLAN_REVIEW

계획 리뷰 스크립트 실행 생략:
- 현재 환경에 `~/.Codex/scripts/codex_plan_review.ps1`가 없다.

수동 검토:
- 사용자의 수정 요청에 따라 72시간이 아니라 24시간 이하만 대상으로 한다.
- 1초마다 `renderTasks()`를 다시 부르면 메모 입력 포커스와 스크롤이 흔들릴 수 있으므로 폭탄 DOM만 업데이트하는 방식이 적절하다.
- 마감 직전 진행률은 `1 - 남은시간/24시간`으로 계산하면 24시간에서 시작, 0시간에서 끝으로 자연스럽게 이동한다.
- D-1 배지는 실제 24시간 초과에도 보일 수 있으므로, D-1 구간에서는 폭탄 레일을 대기 상태로 보여주는 것이 사용자 기대에 맞다.
